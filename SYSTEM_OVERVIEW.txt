PROJECT DESCRIPTION:
-------------------
EduScan is a revolutionary attendance tracking system that combines three
cutting-edge technologies (QR Code, RFID, and Facial Recognition) to provide
flexible, secure, and efficient attendance management for educational
institutions and enterprise organizations.

KEY HIGHLIGHTS:
--------------
✓ Professional eye-catching landing page with animations
✓ 24-day free trial (no credit card required)
✓ Triple technology support (QR Code, RFID, Face Recognition)
✓ Three role-based dashboards (Admin, Teacher, Class Representative)
✓ Real-time analytics and reporting
✓ Multi-tenant architecture
✓ Enterprise-grade security
✓ Mobile-responsive design
✓ Beautiful modern UI with gradients

DATABASE MODELS:
---------------

Organization
  • id, name, type, email, phone, address
  • subscriptionStatus (TRIAL, ACTIVE, EXPIRED)
  • trialStartDate, trialEndDate
  • Relations: users, departments, sessions

User
  • id, email, password, firstName, lastName
  • role (ADMIN, TEACHER, CLASS_REP, STUDENT)
  • rfidTag, faceData, phone
  • departmentId, organizationId, isActive
  • Relations: organization, department, sessions, attendance

Department
  • id, name, code, organizationId
  • Relations: organization, users, courses

Course
  • id, name, code, description, departmentId
  • Relations: department, sessions

Session
  • id, name, courseId, organizationId, creatorId
  • startTime, endTime, location
  • allowedMethods (array), qrCode, qrCodeExpiry
  • isActive, autoCloseOnEnd, lateThreshold
  • Relations: course, organization, creator, attendance

AttendanceRecord
  • id, sessionId, userId
  • status (PRESENT, ABSENT, LATE, EXCUSED)
  • method (QR_CODE, RFID, FACIAL_RECOGNITION)
  • markedAt, location, notes
  • Relations: session, user

Notification
  • id, userId, title, message
  • type (INFO, WARNING, SUCCESS, ERROR)
  • read, createdAt
  • Relations: user

HOW TO GET STARTED:
------------------

QUICK START (Windows):
  1. Double-click start.bat
  2. Follow prompts
  3. Open http://localhost:3000

QUICK START (Linux/Mac):
  1. chmod +x start.sh && ./start.sh
  2. Open http://localhost:3000

MANUAL START:
  1. npm install
  2. cp .env.example .env
  3. Update .env with database credentials
  4. npx prisma generate
  5. npx prisma migrate dev
  6. npm run dev
  7. Open http://localhost:3000

FIRST STEPS:
  1. Sign up for trial at /trial
  2. Check console for admin credentials
  3. Login at /login
  4. Explore admin dashboard
  5. Start building additional features
